## Get the script

Download the script

```bash
wget https://gist.githubusercontent.com/morgbillingsley/1567413fd8d269818cb249cee94c5587/raw/c2ba4cefb13efc9b7cef90b63f8c0a852b16e49a/dylate-deploy.sh
```

Make the script executable

```bash
chmod +x ./dylate-deploy.sh
```

Run the script

```bash
./dylate-deploy.sh
```
